### Searching
- Linear Search

### Sorting
- Bubble / Selection / Insertion Sort

- Quick Sort  
- Counting Sort (when applicable)

### Two Pointers
- Pair sum / triplets  
- Partitioning and rearrangement  
- Palindrome-based problems

### Sliding Window
- Fixed window problems  
- Variable window problems  
- Frequency-based windows

### Recursion & Backtracking
- Subsets / permutations  
- Combination sum patterns  
- N-Queens type problems

### Greedy
- Interval scheduling  
- Minimum/maximum optimization problems  
- Choosing locally optimal decisions

### Dynamic Programming (DP)
- Memoization (Top-down)

- BFS / DFS

### Tree Algorithms
- DFS traversals  
- BFS level order

- LCA and path problems

### Bit Manipulation
- XOR tricks  
- Bit masking  
- Set/unset/check bits

### Math & Number Theory
- GCD / LCM  
- Prime sieve  
- Modular arithmetic  
- Fast exponentiation
